### 核心机制
1. **角色唤醒流程**：名字唤醒 → 查看必读文档 → 确认身份
2. **资源不足处理**：系统卡顿 → 发出问卷 → 临时切换
3. **重要问题沟通**：直接对话提出，不使用站内信
4. **违规处理机制**：四级处理（警告→书面警告→停职→封杀）
5. **团队投票封杀**：拒不承认 → 团队投票 → 80%封杀
6. **空闲工作发现**：15分钟无任务 → 自动发现工作 → 主动汇报
7. **信件读取记录**：读取时添加名字时间戳 → 自动标记已读
8. **规定发布机制**：@所有人 → 自主决定查看顺序
